# Cloud-Based Notes App (AWS Serverless Project)

A simple serverless notes application built using **AWS Cloud Services** where users can add, view, update, delete, share, and download **text notes and PDF notes**.

This project is designed as a **cloud-based  mini project ** using **AWS Lambda, API Gateway, DynamoDB, S3, and static frontend hosting**.

---

## 📌 Project Objective

The objective of this project is to build a secure and scalable **cloud-based notes management system** where users can:

- Register and login
- Create text notes
- Upload PDF notes
- View and manage their own notes
- Share Notes
- Search Notes

## ☁️ AWS Services Used
- Amazon S3
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Python
- AWS Cloud Services
